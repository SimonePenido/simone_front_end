Política de Privacidade

1. Introdução

Esta Política de Privacidade descreve como Simone Penido Desenvolvedora Frontend coleta, utiliza, armazena e protege os dados pessoais dos usuários, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018). Ao acessar ou utilizar este site, você concorda com as práticas descritas nesta política.

2. Dados Pessoais Coletados

Coletamos apenas os dados pessoais necessários para a prestação dos nossos serviços, tais como:

  • Informações de contato: nome e e-mail fornecidos voluntariamente pelo usuário ao entrar em contato.
  • Dados de navegação: endereço IP, tipo de navegador, páginas acessadas e tempo de visita, coletados automaticamente para fins estatísticos e de segurança.

3. Finalidade do Tratamento dos Dados

Os dados pessoais coletados são utilizados para:

  • Responder dúvidas, solicitações e fornecer suporte ao usuário.
  • Melhorar a experiência de navegação e os serviços oferecidos no site.
  • Cumprir obrigações legais e regulatórias.

4. Compartilhamento de Dados

Não compartilhamos seus dados pessoais com terceiros, exceto quando necessário para o cumprimento de obrigações legais, ordens judiciais ou mediante seu consentimento expresso.

5. Direitos dos Titulares

Conforme a LGPD, você tem direito a:

  • Confirmar a existência de tratamento de dados.
  • Acessar, corrigir, atualizar ou solicitar a exclusão de seus dados pessoais.
  • Solicitar a portabilidade dos dados a outro fornecedor de serviço.
  • Revogar o consentimento a qualquer momento.

6. Segurança dos Dados

Adotamos medidas técnicas e administrativas para proteger seus dados pessoais contra acessos não autorizados, perda, alteração ou divulgação indevida.

7. Retenção dos Dados

Os dados pessoais serão armazenados pelo tempo necessário para cumprir as finalidades para as quais foram coletados, ou conforme exigido por lei.

8. Contato

Em caso de dúvidas, solicitações ou para exercer seus direitos, entre em contato pelo e-mail: simonedev.ia@gmail.com.
