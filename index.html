Offered services

Trabalho como desenvolvedora web freelancer há mais de 2 anos. Durante esse tempo, descobri que posso ajudar empresas e empreendedores individuais com os seguintes serviços:

PROJETO

Projetos online de sucesso começam com um bom design. Ele estabelece uma base sólida para o desenvolvimento futuro e permite o crescimento a longo prazo.

DESENVOLVIMENTO

Posso codificar meus próprios designs, utilizar modelos de template ou até mesmo usar o design do cliente como base. Meu foco é gerar código limpo e bem estruturado para confiabilidade.

SEO BASICO

Posso configurar seu projeto para usar princípios básicos de SEO, o que levará seu projeto à primeira página nos mecanismos de busca e poderá ecomomizar recursos.

Contrate-me

Sou uma ótima comunicadora e invisto o tempo necessário para entender o problema do cliente de maneira assertiva.

FERRAMENTAS DE DESIGN

Utilizo as principais ferramentas de design como Color Adobe, Tiny PNG, entre outras para entregar o melhor resultado.

HABILIDADES DE DESENVOLVIMENTO

Estou familiarizada e trabalho diariamente com HTML, CSS, JavaScript, Bootstrap e outras estruturas modernas.
